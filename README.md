# Pharma Assistant 💊

## 📌 About
Pharma Assistant is a web application that helps users search for drugs and find alternatives.

## 🚀 Features
- Drug Search
- Drug Alternatives
- Clean UI using React + MUI

## 🛠️ Tech Stack
- React
- Vite
- Material UI
- React Router

## ⚙️ Run Project
```bash
npm install
npm run dev
